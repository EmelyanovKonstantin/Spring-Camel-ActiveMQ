## About The Project
* Spring-Camel-ActiveMQ boilerplate App.<br>
* Created for training purposes.<br>
* App features:<br>
  * Unmarshal JSON to java object<br>
  * Process object<br>
  * Marshal to XML<br>
  * Send XML to ActiveMQ<br>
